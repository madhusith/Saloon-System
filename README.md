# Salon Management System

Professional web-based salon management system built in phases from the Codex master plan.

## Current Status

The project is in **Phase 1 — Project Foundation**.

This first step creates the repository foundation only. Backend and frontend implementation will be added in later reviewed parts.

## Planned Stack

- React + Vite + JavaScript + Tailwind CSS
- Node.js + Express.js + JavaScript
- MySQL
- JWT authentication
- Socket.IO

## Repository Structure

```text
client/     Frontend React application
server/     Backend Express API
database/   Database schema, seed data, and migrations
docs/       Project documentation
```

## Development Notes

- Do not commit real secrets or passwords.
- Use `.env` files locally and keep `.env.example` files in Git.
- Follow `SALON_MANAGEMENT_SYSTEM_CODEX_MASTER_PLAN.md`.
- Update `TASK_PROGRESS.md` after each reviewed part.
